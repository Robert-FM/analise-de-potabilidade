# Análise de Potabilidade de água
# Stack usada 
Python, Pandas, Numpy, Matplotlib, Jupyter Notebook, Scikit-learn.
# Descrição breve 
Projeto de análise exploratória de dados de parâmetros físico-químicos da água e
análise preditiva de potabilidade.
# Objetivo do projeto 
Realizar EDA do dataset e e prever a potabilidade de água.
# Resultado técnico 
O modelo de Random Forest obteve o melhor desempenho, com acurácia de 78,3%,
F1-score de 0.83 para a classe 0 (não potável) e 0.72 para a classe 1 (potável). O Logistic Regression 
apresentou acurácia de 76,1%, enquanto o KNN ficou com 72,7%.
